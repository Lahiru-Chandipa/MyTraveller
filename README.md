# MyTraveller 🌍

## 🚀 About the Project

MyTraveller is a MERN stack web application for managing travel-related activities.

## 🛠️ Tech Stack

* Frontend: React + TypeScript
* Backend: Node.js + Express
* Database: MongoDB Atlas

## ✨ Features

* User authentication
* API integration
* Clean UI structure

## ⚙️ How to Run

### Backend

cd backend  
npm install  
npm run dev

### Frontend

cd frontend  
npm install  
npm run dev
